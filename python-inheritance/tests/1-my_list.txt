The ``1-my_list`` module
======================

Using ``MyList``
-------------------

Import the class:

    >>> MyList = __import__('1-my_list').MyList

Test empty list:

    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

Test list with one element:

    >>> my_list = MyList([1])
    >>> print(my_list)
    [1]
    >>> my_list.print_sorted()
    [1]

Test list with multiple elements:

    >>> my_list = MyList([1, 4, 2, 3, 5])
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test list with negative numbers:

    >>> my_list = MyList([5, -1, 2, -8, 3])
    >>> print(my_list)
    [5, -1, 2, -8, 3]
    >>> my_list.print_sorted()
    [-8, -1, 2, 3, 5]

Test list with duplicate numbers:

    >>> my_list = MyList([3, 3, 1, 2, 2])
    >>> print(my_list)
    [3, 3, 1, 2, 2]
    >>> my_list.print_sorted()
    [1, 2, 2, 3, 3]

Test that the original list is not modified:

    >>> my_list = MyList([1, 4, 2, 3, 5])
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]
